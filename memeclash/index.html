Game Value

一目で分かる、軽快な3Dバトル。

MemeClashは、複雑な説明を読まなくても遊び始められるテンポを重視しています。短い時間で勝負が動き、キャラの個性で毎回ちょっと違う展開になります。

3Dバトル演出

木製キャラや個性ある相手が、コミカルな動きでぶつかります。

属性と確率

相性、スタン、ギャンブルムーブで、短い対戦にも山場を作ります。

キャラ解放

遊ぶほど新しいキャラクターが増え、次の対戦を試したくなる設計です。

Visual Direction

ゲームらしい勢いを、公式サイトにも。

カジュアルに見える

硬い説明ではなく、キャラクターとステージの絵で遊びの雰囲気を先に伝えます。

短時間プレイ向け

iPhoneで軽く遊ぶゲームとして、テンポ、勝負の変化、キャラの楽しさを中心にしています。

ひと目で伝わる

木製キャラ、バトルステージ、エフェクトを大きく見せ、ゲームの価値を言葉より先に伝えます。
